# Core Data Structure in Pandas.
Pandas mainly works with two staructures.
1- Series - A series is like a single column in Excel.
Example:-
 Index       Value 
 0            10      
 1            20
 2            30
 3            40 
 
Python Example:-
 import pandas as pd 
 data = pd.Series([10,20,30,40])
 print(data)
 
 0 10 
 1 20
 2 30 
 3 40 
 
2- Dataframe
A Dataframe is like an Excel spreadsheet or SQL table.
Example:-
 
 Name      Age    City         Exp (years)
 john      25     Bangalore     8 
 Ravi      28     Delhi         12 
 
 import pandas as pd 
 data= {
      "Name": ["john", "Ravi"],
	  "Age": [25,30,28],
	  "City": ["Bangalore","Delhi"]
	  }
  df = pd.DataFrame(data)
  print (df)
  
    Name   Age    City 
 0  John   25    Bangalore
 1  Ravi   28    Delhi

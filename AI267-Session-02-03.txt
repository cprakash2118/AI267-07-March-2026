CUDA - "Compute Unified Device Architecture".
It is NVIDIA's platform and programming model that let's software use an NVIDIA GPU (Not just of graphics, but for general-purpose computation part).

- CPU => Good at doing a few things very smaertly and sequenentially.
- GPU => Good at doing many semilar calculations at the same time.
- CUDA => Software layer/library ecosystem that helps program to use NVIDIA GPUs for "Acceleration".

FYI- CUDA is the technology stack that "allow developer and frameworks to offload heavy math work from the CPU to the NVIDIA GPU.

FYI - Do not consider this a small libs, it's a very huge eecosystem which includings:-
 - Programming model 
 - Compiler tools 
 - runtime env 
 - drivers 
 - math libraries 
 - ML (deep learning) support libs 

Main idea behind CUDA:- CUDA lets a programmer split work into two part.
A- Host code 
Runs on the CPU 

B- Device code
Runs on the GPU. 

Example:-
 - CPU sends suitable parallel tasks to GPU 
 - GPU processes many opeartions simultaneously and return the outcome.

NOte: Called offloading Computatoin to GPU.

Where CUDA used in AI/ML 
- PyTorch 
- TensorFlow.
